# nirish

niri+ish -> niri as the window manager with the last of the my initials.

# Environment

This config is for `niri` with [noctalia](https://noctalia.dev) shell which is a pre-configured
[QuickShell](https://quickshell.org/) with batteries included.

## Packages required for noctalia to work

1. `wlsunset` - required for night light functionality
